🔥 設定 Firebase 連線

首次使用需輸入你的 Firebase 設定。完成後會儲存在瀏覽器，之後不需重複。

1. 前往 console.firebase.google.com 建立新專案
2. 專案 → 新增應用程式 → 網頁 → 取得 firebaseConfig
3. 左側選單 → Realtime Database → 建立資料庫（選測試模式）
4. 將 firebaseConfig 物件的 JSON 貼入下方
